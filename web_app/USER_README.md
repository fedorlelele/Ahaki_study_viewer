# Ahaki Study Viewer 利用ガイド（ゲスト / 教師）

## ゲストユーザー
- 検索: キーワードで絞り込み。タグは `#タグ名` で検索できます。シリアルは `A01-001` / `a01001` / `A01-001,A02-003` のように検索できます。
- 絞り込み: 科目 / 小項目 / 種別 / 回数で絞り込めます。
- 解答: 選択肢をクリックすると正答・解説が表示されます。
- コピー: 「この問題をコピー」で問題文をコピーできます。
- 要望: タグ / 小項目 / 解説の「修正を要望」からコメントを送信できます（必須）。

## 教師ユーザー
- ログイン: 画面上部の「教師ログイン」からログインします。
- 編集: 解説・タグ・小項目の編集提案を送信できます。
- 承認: 解説を承認して「AI承認済」にできます。

## 更新情報
- 「更新情報」を開くと、解説追加数や機能更新の履歴を確認できます。

## 注意
- ログインしない場合は編集機能は利用できません。
